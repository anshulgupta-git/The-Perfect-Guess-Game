## 📋 About The Project

This is an **engaging number guessing game** where you try to find the secret number! The game provides helpful hints to guide you to the answer.

- 🎮 Try to guess the secret number
- 💡 Get helpful hints (Too High/Too Low)
- 📊 Limited attempts to guess
- 🏆 Score based on attempts used
- 💻 Clean and interactive gameplay

---

## 🎮 How To Play

**Game Rules:**

1. Computer picks a secret number between 1-100
2. You have **limited attempts** to guess it
3. After each guess, you get a hint:
   - 🔼 Too High - Your guess is higher than the secret number
   - 🔽 Too Low - Your guess is lower than the secret number
   - ✅ Perfect - You found the number!
4. Win before your attempts run out!
5. Your score depends on how many attempts you used

## 📦 Installation & Setup

**Prerequisites:**
- Python 3.x installed on your system
- Git (optional)

**Step 1 - Clone the repository:**

    git clone https://github.com/anshulgupta-git/The-Perfect-Guess-Game.git
    cd The-Perfect-Guess-Game

**Step 2 - Run the game:**

    python guess_game.py

**Step 3 - Start guessing!**
